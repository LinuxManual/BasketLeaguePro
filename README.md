# BasketLeaguePro

Professional single-page website for the basketball rivalry between **HotHeroes** and **Ιπτάμενοι**.

## Features
- Large responsive landing page focused on the two teams.
- Team roster management: add members for each team in real time.
- Match management: create games and later store final score.
- Dedicated **Betting Center** section on the site: fans place bets in ευρώ (€), monitor live status, and receive automatic payout calculation (2.00x on win).
- Public Fan Zone chat with Cloud Firestore live updates.

## Run locally
Open `index.html` directly in your browser.

> Note: To use live data (rosters, matches, bets, chat), Firebase/Firestore access must be available in your environment.

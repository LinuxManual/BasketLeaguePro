# BasketLeaguePro

Professional single-page website for the basketball rivalry between **HotHeroes** and **Ιπτάμενοι**.

## Features
- Large responsive landing page focused on the two teams.
- Team roster management: add members for each team and persist data locally.
- Match management: add match date/time/court and save results (scores) later.
- Public Fan Zone chat with Local Storage persistence.

## Run locally
Open `index.html` directly in your browser.
